
okirakuyaho.github.io — Static Site Starter
===========================================

構成
----
/index.html   … HOME
/profile.html … PROFILE
/links.html   … LINKS（配布バナー付き）
/diary.html   … DIARY
/404.html     … GitHub Pages 用
/assets/style.css … 共通スタイル
/images/okiraku_banner.png (+ @2x) … バナー画像

導入手順
--------
1) このZIPを展開（解凍）
2) GitHub のリポジトリ画面で「Add file → Upload files」
3) フォルダごとドラッグ＆ドロップ（assets/ と images/ を含む全ファイル）
4) Commit changes で保存
5) https://okirakuyaho.github.io/ を開いて確認

メモ
----
- バナーを別デザインにしたい場合は images/ のPNGを置き換え（同名でOK）
- 配布コードは links.html の <textarea> に入っています（自動コピー対応ボタン付き）
- デザインは assets/style.css を編集してください
